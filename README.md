The Student Registration Form is a front-end web project developed using HTML5 and CSS3.
This project demonstrates how to design and structure an interactive registration form where users can enter their personal and academic details.

The main goal of this project was to practice building structured forms, applying CSS styling, and creating a clean user-friendly interface for data entry.

🌐 Live Demo

🔗 https://chandrusk3456-tech.github.io/Student-Registration-Form/

🚀 Features

Student information input fields

Personal details section (name, email, phone, etc.)

Gender selection using radio buttons

Course/skills selection using checkboxes

Clean and structured form layout

Submit and reset functionality

Responsive and user-friendly design

🛠️ Technologies Used

HTML5 – Form structure and semantic elements

CSS3 – Styling and layout design

Form Elements – Input fields, radio buttons, checkboxes, dropdowns
